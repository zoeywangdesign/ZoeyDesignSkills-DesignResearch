# Prompt Pack：竞品分析 Design Skill

## Prompt 1：从材料中提炼竞品分析方法论
请从以下竞品调研材料中，提炼出通用的竞品分析方法论。要求：

- 去案例化：不要停留在某一个具体项目背景
- 去产品化：不要过度绑定单一产品或单一行业
- 保留方法性：输出必须体现问题定义、竞品选择、分析维度、方案归纳、自身比较、策略输出
- 保留决策价值：不能只做截图总结，必须支持设计决策

请按以下结构输出：
1. 一句话方法论总结
2. 为什么做竞品分析
3. 标准工作流
4. 通用设计准则
5. 常见误区
6. 可复用模板

---

## Prompt 2：将竞品调研整理成结构化框架
请将以下竞品调研内容整理为一个结构化框架，不要简单复述原文。

请按以下维度展开：
- 问题定义
- 竞品选择逻辑
- 体验目标
- 分析维度
- 典型方案归纳
- 自身位置判断
- 设计策略输出

输出格式优先使用表格 + 条目总结。

---

## Prompt 3：评审一份竞品分析是否合格
请使用“竞品分析 Design Skill”的标准，评审以下竞品分析是否合格。

请重点检查：
- 是否有明确问题定义
- 是否合理筛选竞品
- 是否先定义体验目标再拆解维度
- 是否从具体做法归纳出典型方案
- 是否比较了自身产品位置
- 是否输出了明确设计策略

输出格式：
1. 总体评价
2. 优点
3. 缺失项
4. 风险点
5. 优化建议
6. 最终评分（满分10分）

---

## Prompt 4：把零散竞品截图整理成方法论
以下是一些零散的竞品截图与说明，请不要仅按产品逐个罗列，而是整理成一套支持设计决策的竞品分析方法结论。

要求：
- 先定义问题
- 再识别体验目标
- 再归纳方案类型
- 再判断自身位置
- 最后输出策略
